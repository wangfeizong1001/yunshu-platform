# 更新日志

所有重要的项目更新都会记录在此文件中。遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.0.0] - 2024-XX-XX

> 发布日期：2024-XX-XX

### ✨ 新功能

#### 设计系统

- **设计令牌系统** - 支持 CSS/SCSS/JS/Tailwind 多格式输出
- **双主题支持** - 浅色/深色主题，开箱即用
- **WCAG AA 标准** - 遵循无障碍对比度标准

#### 权限管理

- **RBAC 权限模型** - 基于角色的访问控制
- **按钮级权限** - 精确到按钮的权限管理
- **数据权限控制** - 细粒度的数据访问控制
- **多租户 SaaS** - 多租户架构支持

#### 系统功能

- **用户管理** - 完整的用户 CRUD 操作
- **角色管理** - 角色创建、编辑、分配权限
- **菜单管理** - 动态菜单配置
- **操作日志** - 记录用户关键操作
- **登录日志** - 完整的登录审计

#### 基础设施

- **Monorepo 架构** - 使用 Turborepo 管理多包仓库
- **TypeScript 严格模式** - 完整的类型安全
- **动态路由** - 根据权限自动生成菜单
- **Mock 数据服务** - 无需后端即可开发
- **国际化支持** - 中英文切换

#### 组件库

- **YunButton** - 按钮组件，支持多种类型和状态
- **YunInput** - 输入框组件
- **YunForm/YunFormItem** - 表单及表单项组件
- **YunTable** - 表格组件，支持分页和自定义列
- **YunDialog** - 对话框组件
- **YunTag** - 标签组件
- **YunSelect** - 选择器组件

#### API 客户端

- **请求拦截器** - 请求前后处理
- **响应拦截器** - 统一错误处理
- **Token 刷新** - 自动刷新过期 Token
- **请求重试** - 失败请求自动重试
- **请求缓存** - 支持请求结果缓存
- **请求去重** - 防止重复请求

#### 后端核心

- **BaseService** - 服务层基类
- **装饰器体系** - @Get, @Post 等路由装饰器
- **错误处理** - 统一错误处理体系
- **Express 适配器** - Express 框架适配

### 🐛 修复

- 修复了若干已知问题

### 📝 文档

- 新增完整的 README.md 文档
- 新增 CONTRIBUTING.md 贡献指南
- 新增 DEVELOP.md 开发指南
- 新增 DEPLOY.md 部署指南
- 新增 API.md API 文档
- 新增 OPENAPI.md OpenAPI 规范

### 🏗️ 构建

- 升级 Turborepo 到 v1.13.0
- 升级 TypeScript 到 v5.4.0
- 升级 pnpm 到 v9.0.0
- 升级 Node.js 要求到 >= 18.0.0
