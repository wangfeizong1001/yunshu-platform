## [0.2.0] — 2025-06-09

### Added

- ✨ **后台管理应用** — 基于 Vue 3 + Element Plus 的完整管理后台
- ✨ **用户管理模块** — 用户 CRUD、角色分配、密码重置、部门归属
- ✨ **角色权限模块** — RBAC 权限模型、菜单权限、按钮级权限
- ✨ **部门管理模块** — 树形组织架构、嵌套层级管理
- ✨ **菜单管理模块** — 动态菜单配置、路由元信息管理
- ✨ **岗位管理模块** — 岗位定义与用户分配
- ✨ **字典管理模块** — 系统字典与枚举值管理
- ✨ **参数配置模块** — 系统参数动态配置与热更新
- ✨ **文件管理模块** — 文件上传、下载、预览、管理
- ✨ **表单设计器** — 拖拽式可视化表单构建
- ✨ **知识库系统** — 文档创作、分类管理与全文检索
- ✨ **消息中心** — 站内消息发送、接收与已读标记
- ✨ **通知公告** — 系统通知发布与多端推送
- ✨ **OSS 对象存储** — 多存储源适配（本地/阿里云/七牛/腾讯云）
- ✨ **工作流引擎** — 流程设计器、待办/已办任务、流程实例管理
- ✨ **报表引擎** — 报表设计、预览、导出（Excel/PDF）
- ✨ **大屏看板** — 企业级数据可视化大屏
- ✨ **多租户架构** — 租户管理、套餐配置、数据隔离
- ✨ **SSO 单点登录** — 标准协议接入（CAS/OAuth2/SAML）
- ✨ **短信服务** — 多供应商短信服务集成与模板管理
- ✨ **第三方登录** — 微信、钉钉、企业微信等一键登录
- ✨ **系统监控模块** — 操作日志、登录日志审计
- ✨ **在线用户监控** — 实时会话管理与强制下线
- ✨ **服务监控** — 系统健康状态与资源实时监控
- ✨ **定时任务** — Cron 表达式任务调度与手动触发
- ✨ **代码生成器** — 一键生成前后端 CRUD 代码
- ✨ **Monorepo 架构** — 使用 Turborepo 管理多包仓库
- ✨ **TypeScript 严格模式** — 完整的类型安全保障
- ✨ **国际化系统** — 中英文切换、可扩展多语言
- ✨ **Mock 数据系统** — 完整业务场景模拟，无需后端即可开发
- ✨ **结构化日志** — Winston + Loki 集中收集与分析
- ✨ **缓存与分布式锁** — Redis 二级缓存 + 布隆过滤器防穿透

- ✨ **上传安全校验** — 文件类型白名单、魔数校验、大小限制

## 版本规范

### 语义化版本号规则

- **主版本号 (MAJOR)** — 不兼容的 API 变更
- **次版本号 (MINOR)** — 向后兼容的功能性新增
- **修订版本号 (PATCH)** — 向后兼容的问题修复

### 预发布版本标识

- `alpha` — 内部测试版，功能可能不完整
- `beta` — 公开测试版，功能基本完整但可能有 Bug
- `rc` — 发布候选版，功能冻结，仅修复严重 Bug

**示例**：`v2.0.0-alpha.1`、`v2.0.0-beta.3`、`v2.0.0-rc.2`
