## 🚀 快速开始

### 方式一：Docker Compose（推荐）

```bash
# 1. 克隆项目
git clone https://github.com/wangfeizong1001/yunshu-platform.git
cd yunshu-platform

# 2. 一键启动
docker-compose up -d

# 3. 访问应用
# 前端: http://localhost
# 后端: http://localhost:8080
```

### 方式二：本地开发

```bash
# 1. 安装依赖
pnpm install

# 2. 启动开发服务器
cd apps/admin
npm run dev

# 3. 运行测试
npm run test

# 4. 构建生产版本
npm run build
```
